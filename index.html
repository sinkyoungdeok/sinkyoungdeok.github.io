Project(Detail)

AI 기사 수집기 with POSCO int.

- 사원에게 필요한 뉴스기사를 메일로 전송하는 예약 서비스 -
소개
몇몇 데이터 기준만 입력하면 AI가 사원 개개인의 취향에 맞춰 뉴스기사를 수집하여 예약된 시간마다 해당 기사를 개인 e-mail로 전송해주는 서비스
예를 들면,
  1. 사원이 아래 데이터를 입력
    • 키워드: 베트남
    • 크롤링 할 포털사이트: naver
    • 뉴스 기사 페이지수: 10page (1page 당 10개씩, 100개)
    • 크롤링 할 날짜: 일주일 전 까지
    • 전달 받을 유사한 뉴스 기사들의 갯수: 5개 (랭킹 5위까지)
    • 전달 받을 메일: tkagmd1@naver.com
    • 예약 시간: 매일 오후 2시
  2. 예약 시간이 되면 베트남 키워드로 10page만큼 일주일 전 까지의 naver 뉴스 기사들을 크롤링 해서 DB에 저장
  3. DB에 저장 되어 있는 뉴스기사들과 그동안 유저가 관심(좋아요)기사로 추가 하였던 뉴스기사들을 불러와서 벡터화를 시킨 후 유사도 검사를 진행
  4. 유사도 점수에 따라서 랭킹화를 하여 랭킹 5위까지를 tkagmd1@naver.com로 뉴스기사 url을 보냄
상세 내용
  • 요구사항: 매일 포털사이트(네이버, 구글 등)에 접속하여 25개의 국가(베트남, 미국 등), 7개의 품목(환율, 국제유가 등)을 검색하여 회사의 필요에 적합한 뉴스기사들을 저장해두는 작업을 자동화 혹은 서포트 해줄 수 있는 시스템 구축
  • 상황 분석
    • 시스템 사용자 수가 정해져 있지 않음
    • 유저 개개인의 다양한 니즈를 포함해야 함
  • 역할
    • 웹사이트 풀스택 개발
      - 로그인 기능을 통해 프로그램 사용자 수 변화에 대응
    • 네이버·구글 크롤러 제작
    • 추천 기능 구축
      - 유저 개개인의 다양한 취향을 모두 적용하기 위해 좋아요 기능을 기획과 구현
    • 예약 기능을 제안과 구현
