AI 기사 수집기 with 포스코 인터내셔널

요구사항: 사원마다 필요한 뉴스 기사를 매일 수집하여 저장하는 업무 자동화

설계요약: 사용자의 필요한 뉴스 기사 혹은 키워드 등을 입력 받는다. 그 정보를 활용해서 여러 다른 뉴스 기사들과의 유사도 검사를 통해 점수를 매긴다. 그리고 랭킹화하여 높은 순위의 뉴스기사들을 선별하여 각 사용자의 관심 있을 법한 뉴스기사들을 메일로 전달해준다.

사용 기술
웹사이트 - python (flask, jinja2)
크롤링 - python (beautifulsoup, selenium)
딥러닝모델(임베딩) - word2vec, tf-idf
유사도 검사 - cosine

Role: 풀스택 개발, 인공지능 개발, 프로젝트 팀장
